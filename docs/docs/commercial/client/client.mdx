# Fiche Client

<CustomCard img={"/img/commercial/customer_focus.png"} />

La **fiche client** est l'une des fonctionnalités clés du module **"Commercial"**. Elle permet de visualiser toutes les informations relatives à un client enregistré dans l'application. La fiche client se décompose en **4 parties** :

- Les coordonnées de contact du client sélectionné,
- Les dernières informations en lien avec les relations commerciales :
  - Commercial référent,
  - Dernière visite,
  - Objectif de productions de dossiers,
- Les données en lien avec la facturation et la comptabilité :
  - Statut de paiement,
  - Objectif de facturation,
- Les comptes-rendus de visite / réunion

Cette vue globale permet aux commerciaux de mieux connaître leur client et de suivre leur historique de relations commerciales de manière efficace.
