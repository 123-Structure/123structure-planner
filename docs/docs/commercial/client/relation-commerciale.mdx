# Relation Commerciale

## Informations générales

### Mode lecture

La carte **Relation commerciale** affiche plusieurs informations :

- Le / Les commerciaux référent(s),
- Date et objet de la dernière visite / réunion,
- Objectif de production pour l'année n-1 avec le client sélectionné,
- Production de l'année n-1 avec le client sélectionné,
- Objectif de production pour l'année cours avec le client sélectionné,
- La mise en place d'un contrat cadre si visible,
- Grille tarifaire au format PDF

:::info

La production de l'année passée et de l'année en cours sera récupérée depuis le module **[Planning](/docs/category/planning)** une fois mis en place.

:::

Dans ce mode, vous pouvez modifier les informations suivantes :

- Le / Les commerciaux référent(s),
- L'objectif de production pour l'année n-1 avec le client sélectionné,
- L'objectif de production pour l'année an cours avec le client sélectionné,
- La mise en place d'un contrat-cadre,
- Le ficher "Grille tarifaire" **au format PDF**
