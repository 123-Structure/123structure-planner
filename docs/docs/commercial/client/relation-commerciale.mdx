# Relation Commerciale

<CustomCard img={"/img/commercial/customer_relationship.png"} />

## Informations générales

### Mode lecture

La carte **Relation commercial** affiche plusieurs informations :

- Le / Les commerciaux référent(s),
- Date et objet de la dernière visite / réunion,
- Objectif de production pour l'année n-1 avec le client séléctionné,
- Production de l'année n-1 avec le client séléctionné,
- Objectif de production pour l'année cours avec le client séléctionné,
- Production de l'année en cours avec le client séléctionné,
- Grille tarifaire au format PDF

:::info

La production de l'année passée et de l'année en cours sera récupérée depuis le module **[Planning](/docs/category/planning)** une fois mis en place.

:::
