## Application

<CustomCard img={"/img/commercial/commercial.png"} />

C'est le corps du module **"Commercial"**. C'est ici que vous retrouvez les menus de filtrage menant vers les clients enregistrés dans l'application. Vous pouvez ainsi accéder aux informations suivantes :

- Les coordonnées d'un client (adresse postale, numéro de téléphone, ...),
- Les interlocuteurs,
- Le commercial référent,
- Le nombre de dossiers produits et la facturation,
- Les comptes-rendus de visite,
- ...

Le module **"Commercial"** vous permet également de créer de nouveaux clients, de gérer les dossiers commerciaux et de suivre l'évolution des ventes. Nous vous présentons en détail toutes les fonctionnalités du module **"Commercial"** dans les sections suivantes.
