## Application

  <div class="card__body shadow--md">

  </div>

C'est le corps du module "Commercial". C'est ici que vous retrouvez le carnet d'adresses vers les clients enregistrés dans l'application avec nottament :

- Les coordonées d'un client (adresse postale, numéro de téléphone, ...),
- Les interlocuteurs,
- Le commercial référent,
- Le nombre de dossiers produits et la facturation,
- Les comptes-rendus de visite,
- ...
